BRIEF DESCRIPTION OF FLYFLEX FLIGHT PROJECT

I designed this system to streamline the process of booking and managing flights through a
user-friendly interface. The platform offers a simple homepage with clear login and signup options.
For new users, registration is straight forward while returning users can log in and immediately access
the user's dashboard. Once inside, users can search for available flights, book them instantly and view a
personalized overview of their bookings and reports. When they are done, they can securely logout of the system.

FlyFlex also features an admin section. When the admin logs in, they are directed to the admin dashboard
which offers powerful management tools. Admins can monitor user bookings, apply filters to analyse data and
download comprehensive reports.

The project aims to offer efficient, transparency and accessibility for both users and admins in the flight booking process.
